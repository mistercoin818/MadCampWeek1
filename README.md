## 📱Application 개요
2023 KAIST 여름 몰입캠프 공통과제 1(2023.06.29. ~ 2023.07.05.)을 수행한 결과물입니다.


코로나 양성 두 명과 음성 한 명으로 이루어진 팀이라 Application 이름을 Helium+로 지었습니다.


총 3개의 탭으로 이루어져 있습니다.
1. 탭1: 나의 연락처
2. 탭2: 나만의 이미지 갤러리
3. 탭3: 위치 기반 날씨 정보
